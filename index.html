CROSS COUNTRY RACING

DOES PRACTICE MAKE PERFECT?

LET'S FIND OUT!

This website shows eight years of high school cross country race data analysis of 63,789 runners from 314 schools running 1,211 courses in 9033 races and nearly two million training log entries. Select the NEXT button at the bottom of each page to see the controllable and uncontrollable factors that influence race performance. Alternatively, select specific factors from the GRAPHICAL ANALYSIS drop-down menu above.

Information about project architecture, machine learning importances, and the data generously provided by XCStats can be navigated to from the MACHINE LEARNING & DATA drop-down menu above.
